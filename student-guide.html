📚 학생용 사용 가이드

UTTEC ESP32 IoT 교육 시스템 사용법

💬 질문하는 2가지 방법

📌 언제 무엇을 사용하나요?
  • 실시간 채팅: 수업 시간에 바로 답변이 필요할 때
  • 이메일 질문: 수업 후 또는 복잡한 질문을 자세히 쓸 때

💬 방법 1: 실시간 채팅으로 질문하기

✅ 언제 사용하나요?
  • 수업 시간에 선생님이 온라인일 때
  • 간단한 질문이나 빠른 확인이 필요할 때
  • 실습 중 막혔을 때 즉시 도움이 필요할 때

사용 방법:

1 웹사이트 접속

브라우저에서 http://3.35.139.224 접속

2 채팅 버튼 찾기

화면 오른쪽 아래에 있는 보라색 버튼을 찾아요

💬 선생님께 질문

3 채팅창 열기

버튼을 클릭하면 채팅창이 나타납니다

4 연결 확인

채팅창 위쪽에 "✅ 연결 완료! 이름을 입력하세요." 라는 메시지가 나올 때까지 기다려요 (보통 2-3초)

5 이름 입력

자신의 이름을 입력하고 Enter 키를 눌러요

⚠️ 중요: 이름을 입력한 후 꼭 Enter 키를 눌러야 합니다!

6 질문 메시지 입력

아래 입력창이 활성화되면 질문을 입력하고 "전송" 버튼을 클릭하거나 Enter 키를 눌러요

7 선생님 답변 기다리기

선생님이 채팅을 보고 있으면 곧 답변이 올라와요!

⚠️ 채팅이 안 될 때
  • "연결 오류"가 나오면: 페이지를 새로고침(F5 키) 하세요
  • 이름 입력 후 메시지 창이 안 열리면: Enter 키를 눌렀는지 확인하세요
  • 선생님이 답변을 안 하면: 선생님이 지금 바쁠 수 있어요. 이메일 질문을 이용하세요!

✉️ 방법 2: 이메일로 질문하기

✅ 언제 사용하나요?
  • 수업이 끝난 후 집에서 질문할 때
  • 복잡한 문제를 자세히 설명하고 싶을 때
  • 선생님이 오프라인일 때
  • 천천히 생각해서 답변 받고 싶을 때

사용 방법:

1 질문 페이지 열기

메인 페이지에서 주황색 버튼을 클릭해요

✉️ 이메일 질문하기

2 양식 작성하기

다음 정보를 입력해요:

  • 이름: 본인의 이름
  • 이메일: 답변 받을 이메일 주소 (정확히!)
  • 카테고리: Week 1, 2, 3 또는 기타 선택
  • 질문 제목: 질문을 한 줄로 요약
  • 질문 내용: 자세한 설명
3 질문 제출

"✉️ 질문 제출" 버튼을 클릭해요

4 확인 메시지

"✅ 질문이 성공적으로 제출되었습니다!" 메시지가 나오면 완료!

5 답변 이메일 기다리기

선생님이 답변을 작성하면 입력한 이메일로 답변이 옵니다

💡 좋은 질문 작성하는 방법
  • 구체적으로: "Week 1 Day 2의 LED 켜기 실습에서..."
  • 무엇을 했는지: "코드를 정확히 입력하고 업로드했는데..."
  • 무엇이 문제인지: "LED가 켜지지 않아요"
  • 에러 메시지: 있다면 정확히 복사해서 붙여넣기

좋은 질문 예시:

질문 제목: Week 1 Day 2 LED가 켜지지 않아요

질문 내용:
안녕하세요! Week 1 Day 2의 LED 켜기 실습을 하고 있는데 문제가 있어요.

무엇을 했나요:
- 교재 31페이지의 코드를 Arduino IDE에 정확히 입력했어요
- LED를 GPIO 2번 핀에 연결했어요
- 업로드도 성공했다고 나와요

무엇이 문제인가요:
- 프로그램을 실행해도 LED가 전혀 켜지지 않아요
- 시리얼 모니터에는 아무것도 안 나와요

도와주세요!

나쁜 질문 예시:

질문 제목: 안돼요

질문 내용: LED가 안 켜져요

❌ 이렇게 쓰면 선생님이 어떤 부분을 도와줘야 할지 모릅니다!

📖 교육 자료 보는 방법

1 웹에서 학습하기

메인 페이지에서 파란색 "📖 웹에서 학습하기" 버튼 클릭

→ Week 1, 2, 3 중에서 선택해서 공부해요

2 자료 다운로드

집에서 인터넷 없이 공부하고 싶다면?

초록색 "💾 전체 자료 다운로드" 버튼 클릭

→ ZIP 파일을 다운로드하고 압축 풀기

→ index.html 파일을 더블클릭하면 브라우저에서 열려요
